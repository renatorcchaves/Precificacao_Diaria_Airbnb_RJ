DADOS DO PROJETO

- As bases de dados foram retiradas do site kaggle: https://www.kaggle.com/allanbruno/airbnb-rio-de-janeiro

- Através do arquivo o arquivo "notebooks/df_projeto_airbnb_rj.ipynb" foi importado os dados que foram obtidos pelo Kaggle, realizado a análise exploratório e o tratamento dos dados para o modelo de machine learning, e extraído o arquivo "dados_finais.csv".

- Esse arquivo "dados_finais.csv" está sendo usado pelo arquivo "notebooks/deploy_projeto_airbnb_rj.py" para prever o valor da diária de um airbnb no Rio de Janeiro pela plataforma do Streamlit.

- Caso queiram fazer essa previsão do valor da diária de um imóvel, precisa fazer o dowload do dados do Kaggle, executar o arquivo "notebooks/df_projeto_airbnb_rj.ipynb" para extrair o modelo de machine learning e os "dados_finais.csv" do projeto, e executar o arquivo "notebooks/deploy_projeto_airbnb_rj.py" em um terminl do windows, e abrir o link do Streamlit inputando as informações necessárias.
